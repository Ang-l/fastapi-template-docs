#  FastAPI Template

FastAPI Template 是一个基于 **FastAPI** 框架构建的高性能 Web 应用程序项目模板。它提供了一套结构清晰、易于扩展的项目架构，集成了常用的功能模块与行业最佳实践。

该模板旨在帮助开发者跳过繁琐的基础设施搭建过程，让你能够专注于核心业务逻辑的实现，快速构建生产级的后端服务。

---

##  核心特点

- **️ 快速启动**：提供开箱即用的预配置项目结构，无需从零开始搭建，极大缩短项目初始化时间。
- ** 极致性能**：深度利用 FastAPI 的异步特性（ASGI）与高性能设计，轻松应对高并发请求，提供毫秒级响应。
- ** 模块化设计**：采用清晰的模块化分层架构，解耦业务逻辑与基础设施，便于后期的功能扩展与维护。
- **️ 内置常用功能**：集成了用户认证、数据库连接（ORM）、线程安全日志、统一异常处理等基础模块，拒绝重复造轮子。
- ** 友好的开发体验**：配合完善的文档结构与示例代码，结合 FastAPI 原生的 Swagger UI，接口调试更加直观。
- ** 社区驱动**：基于成熟的开源生态构建，拥有活跃的社区支持，便于获取技术帮助与分享最佳实践。

::: tip  提示
该模板特别适合需要快速交付、且对性能和代码规范性有较高要求的中大型后端项目。
:::

## ️环境要求与准备

在开始之前，请确保你的开发环境满足以下要求：

- **Python**: `3.11+` (本项目推荐使用 Python 3.11)
- **操作系统**: Windows / macOS / Linux 均可完美运行
- **包管理器**: 推荐使用 `conda` 或 `pip` 进行依赖管理

### ️ 快速创建虚拟环境（推荐）

为了保持项目依赖的纯净与隔离，建议使用 Conda 创建一个专属的虚拟环境。你可以在终端中执行以下命令：

```bash
# 创建名为 ft 的 Python 3.11 虚拟环境
conda create -n ft python=3.11

# 激活该环境
conda activate ft
```

::: tip 提示
成功激活 ft 环境后，你将看到终端前缀变为 (ft)。此时再进行后续的依赖安装和项目启动，就能确保所有配置都在这个独立的环境中运行啦！
:::


## 项目结构
项目采用了经典的分层架构设计，各目录职责明确：

```text
backend/
├── controllers/            # [控制层] 接口业务逻辑处理
│   ├── __init__.py 
│   └── health.py           # 示例：健康检查接口
│
├── infrastructure/         # [基础设施层] 外部依赖配置
│   ├── __init__.py
│   ├── config.py           # 全局配置文件加载
│   └── database.py         # 数据库连接与会话管理
│
├── libs/                   # [公共组件库] 基础服务与工具类
│   ├── __init__.py
│   ├── security/           # 安全相关组件
│   │   ├── __init__.py
│   │   ├── password.py     # 密码哈希与校验
│   │   ├── token.py        # JWT Token 生成与解析
│   │   └── scheme.py       # API 文档安全方案配置
│   │
│   ├── authenticate.py     # 全局认证服务
│   ├── code.py             # 统一业务状态码定义
│   ├── constants.py        # 全局常量定义
│   ├── decorators.py       # 自定义装饰器
│   ├── exception.py        # 自定义异常类
│   ├── headers.py          # 请求头处理工具
│   ├── loader.py           # 动态加载工具
│   ├── logger.py           # 线程安全的日志封装
│   ├── resource.py         # 资源管理
│   └── response.py         # 统一 API 响应格式封装
│
├── models/                 # [数据模型层] ORM 模型定义
│   ├── __init__.py
│   ├── base.py             # 数据库基类
│   ├── user.py             # 用户表模型
│   └── role.py             # 角色表模型
│
├── urls/                   # [路由层] URL 路由分发配置
│   ├── __init__.py
│   ├── router.py           # 主路由注册
│   └── health.py           # 健康检查路由
│
├── migrations/             # [数据库迁移] Alembic 迁移脚本目录
├── logs/                   # [日志存储] 运行时日志文件输出目录
│
├── .env                    # 本地环境变量配置
├── .env.dev                # 开发环境变量配置
├── .env.prod               # 生产环境变量配置
├── .gitignore              # Git 忽略文件配置
├── alembic.ini             # Alembic 数据库迁移配置文件
├── env.py                  # Alembic 环境配置入口
├── main.py                 # FastAPI 应用主入口
├── run.py                  # 项目启动脚本
└── README.md               # 项目说明文档

```

# ️准备


在开始之前，请确保你的开发环境满足以下要求：

- **Python**: `3.11+` (本项目推荐使用 Python 3.11)
- **操作系统**: Windows / macOS / Linux 均可完美运行
- **包管理器**: 推荐使用 `conda` 或 `pip` 进行依赖管理

### ️ 快速创建虚拟环境（推荐）

为了保持项目依赖的纯净与隔离，建议使用 Conda 创建一个专属的虚拟环境。你可以在终端中执行以下命令：

```bash
# 创建名为 ft 的 Python 3.11 虚拟环境
conda create -n ft python=3.11

# 激活该环境
conda activate ft
```

::: tip 提示
成功激活 ft 环境后，你将看到终端前缀变为 (ft)。此时再进行后续的依赖安装和项目启动，就能确保所有配置都在这个独立的环境中运行啦！
:::
